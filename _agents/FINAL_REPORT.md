## Следующие шаги

1. Установить зависимости: `npm install`
2. Запустить dev сервер: `npm run dev`
3. Протестировать функциональность
4. При необходимости настроить переменные окружения для API ключей
5. Выполнить production сборку: `npm run build`

## Заметки

- Handlebars шаблоны используют синтаксис `{{> partials/header}}` в index.html
- Vite плагин для Handlebars должен автоматически обрабатывать эти шаблоны
- API ключи Airtable можно вынести в .env файл для production
- Все статические файлы должны быть в папке `public/`
